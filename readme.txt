ARQUIVO INDEX
.....................................................................................................................
O arquivo index.php é responsável por executar o arquivo autoload, que por sua vez carrega todas as bibliotecas e classes a serem utilizadas no projeto
.....................................................................................................................


ARQUIVO .htaccess
.....................................................................................................................
Arquivo responsável por fazer funcionar as rotas do projeto
.....................................................................................................................


RIRETÓRIO VENDOR
.....................................................................................................................
Diretório responsável por guardar todas as bibliotecas locais e também o diretório MF que foi utilizado para guardar classes criadas para uso de forma global como uma biblioteca Bootstrap por exemplo.
.....................................................................................................................


BIBLIOTECAS UTILIZADAS NO PROJETO
.....................................................................................................................
Composer - Bootstrap - JQuery  - Popper - Fontawesome

As bibliotecas Popper - Fontawesome não foram baixada no projeto, sendo utilizadas diretamente pelo link oferecido.
.....................................................................................................................


DIRETÓRIO APP
.....................................................................................................................
Diretório utilizado para guardar todas os arquivos da arquitetura MVC
Para isso foram criados os diretórios Controlles - Models - Views
.....................................................................................................................


ARQUIVO Routes.php
.....................................................................................................................
Criado para controlar todas as rotas do projeto. Ele está localizado dentro do diretório App
.....................................................................................................................


DIRETÓRIO SRC
.....................................................................................................................
Criado para guardar arquivos css e imagens utilizadas no projeto
.....................................................................................................................


ARQUIVO banco_vitallife.sql
.....................................................................................................................
Criado com as configurações utilizadas para criar o banco e tabelas que foram utilizadas no projeto. Foi utilizado o banco MariaDB (MySql).
.....................................................................................................................


CONTEÚDO DOS ARQUIVOS
.....................................................................................................................
Foram criados comentários dentro de cada arquivo, facilitando manutenção do projeto futuramente.
